## 🎯 核心思路

```
注册阶段 → 真实美国家宽 IP（Mysterium）
日常使用 → 干净稳定的代理/VPS
关键原则 → 注册 IP ≠ 日常 IP 可以，但不能来回跳
```

## 🔧 环境

- 主机：Windows
- 日常 VPN：魔戒（机场）
- 注册 IP：Mysterium（P2P 家宽网络）
- 手机验证：HeroSMS（美国物理手机号）
- 支付：美区 Apple ID + 礼品卡 → App Store 内购 Claude Pro
